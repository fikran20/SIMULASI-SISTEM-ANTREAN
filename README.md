# SIMULASI SISTEM ANTREAN

Proyek ini dibuat untuk memenuhi Tugas 3 Mata Kuliah Rekayasa Perangkat Lunak (RPL) - Kelas 6B. Program ini berfungsi untuk mensimulasikan mekanisme antrean (queueing system) guna menganalisis kinerja pelayanan berdasarkan parameter waktu kedatangan dan waktu pelayanan pelanggan.

# IDENTITAS MAHASISWA

Nama: Fikran

Kelas: RPL - 6B

Tugas: Tugas 3 (Simulasi Sistem Antrean)

# FITUR PROGRAM

Simulasi Kedatangan Antrean: Menghitung waktu antar-kedatangan pelanggan secara acak atau berdasarkan distribusi tertentu.

Metrik Kinerja: Mengukur waktu tunggu rata-rata pelanggan dalam antrean, waktu total dalam sistem, serta tingkat utilitas pelayan (server).

Log Antrean: Menampilkan urutan antrean dan status pelayanan secara terstruktur melalui terminal.

# TEKNOLOGI DAN BAHASA PEMROGRAMAN

Bahasa: Python

Library: Standard Library (Queue, Random, Time)
